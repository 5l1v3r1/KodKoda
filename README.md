## Frontend

Get familiar with Material-UI and checkout its API with [mui](https://material-ui.com/getting-started/learn/)

To run the client website for development

```
cd client
npm install
npm run dev
```

To run the client website for production

```
cd client
npm install
npm run build
npm run start
```
